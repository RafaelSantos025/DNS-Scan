# DNS-Scan Description
DNS-Scan is a simple python tool designed to find subdomains using bruteforce.

## Dependencies

* <b>host</b>:
    ```sudo apt install host```
    
* <b>Python</b>
    ```sudo apt install python```

## Setup

```chmod +x dns_scan```

## How To Use

```./dns_scan <domain> <wordlist>```
